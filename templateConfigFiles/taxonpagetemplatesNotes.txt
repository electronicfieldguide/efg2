Convention for displaying templates on jsp pages.
See TaxonPageTemplate.xsd

DataTypes:

1.groups
	Is the root node. It can never be a leaf node. It has children 'group'.
	It could be recursive. 
	e.g:
		groups->group->character or groups->group->groups->group->character

2.group:
	Has groups or character as children. It can never be a leaf node.

3. characterValue
	Is always a leaf node.


4.gl is a label type- means a group label  

5. cl is a label type -means a character label 

6. tl is a text label for a group or character - allows users to specify the text that should be used

7. text is a piece of text that should be used as replacement for a state/pseudo state


Convention for naming things in jsp template pages
gl means group label
cl means character label

label : group id : group rank:character rank 
'gl':groupid:group rank[:character rank]
'cl':groupid:group rank:character rank
'tl':groupid:group rank[:character rank]



Things with the same group rank will be sorted based on the rank of their characterValues.

	<group rank="1" id="1">
	       <characterValue rank="1" text="Field Guide to the Bogs of Southern New England"/>
	        <characterValue rank="2" text="by Marsha Salett"/>
	</group>
