Follow these instructions after installation.

Note: $INSTALL_PATH is where the application was installed.
$catalina_home is the full path to where your Tomcat server was installed.

  • Look in $catalina_home/webapps and make sure efg2.war is present. If it is not, physically copy $INSTALL_PATH/resource/efg2.war to that location or run $INSTALL_PATH/deploywebapps.sh. You may need to set the necessary file permissions to execute the script.
  • Restart Tomcat
  • Run $INSTALL_PATH/efgImport.sh to start up the EFG2 Data Importer
  • To uninstall application, execute java -jar $INSTALL_PATH/Uninstaller/uninstall.jar
  • For further instructions see Manual
