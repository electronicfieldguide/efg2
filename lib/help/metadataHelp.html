EFG2 GUI Help - Data Source Configuration
Searchable Check this box if you want the field to appear on the search page
Lists Check this box if the field contains list data (Similar Species, Associates, etc.)
Taxon Page Check this box if you want the field to appear on the Taxon Pages
Categorical Select if the field contains categorical states (red, green, blue etc.)
Narrative Select if field contains any descriptive data (sentences, phrases, etc.). If the field contains a mix of data types you must select Narrative.
Numeric Value Select to indicate that field contains discrete numbers (2,4,5 etc.)
Numeric Range Select to indicate that field contains numeric ranges (10-20, 20-30, etc.
Media Resource Select to indicate that field holds media resources (images,audio, video, etc.)
Order Determine the order of fields on the search page by dragging rows to the desired position ('3' means it will appear as the 3rd field on the search page)
Update Button Click to apply your changes
Close Button Close the metadata editing window
