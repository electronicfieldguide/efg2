IsSearchable:

  • Check for field and contents to appear on search page.

isLists:

  • Check to indicate that the field holds list data

OnTaxonPage:

  • Check for field and contents to appear on Taxon Page

Categorical:

  • Select to indicate if character has categorical states

NumericValue:

  • Select to indicate that field holds atomicized numeric values 2,4,5 etc

NumericRange:

  • Select to indicate that field holds numeric ranges. For eg 10-20,20-30 etc

MediaResource:

  • Select to indicate that field holds media resources. For eg images,audio or video

OrderValue:

  • Indicate the position of field on search page, For eg 3 means it will appear as the 3rd field on the search page

Update Button:

  • Click to update the selections you have made

Close Button:

  • To close window
