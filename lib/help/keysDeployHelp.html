Import Data:

  • From your desktop select the comma separated data file you wish to import.
  • Drag and drop it anywhere in the window on the left hand side.
  • Wait for Application to indicate whether import was successful or not.

View/Edit Metadata Table:

  • Select a datasource from the list in the left window
  • Then click on the 'View/Edit Metadata Table button'
  • Edit the resulting table to fit your data source

Delete Datasource:

  • Select a datasource from the list in the left window
  • Then click the 'Delete Datasource' button to delete it.
  • Note that you cannot undo this action, so think hard before you delete a datasource.

Change Display Name:

  • Select a datasource from the list in the left window
  • Then click on the 'Change Display Name ' button to change the display name
  • Note that the display name is the name that the application will use to display your data source

Done:

  • To close the current window
