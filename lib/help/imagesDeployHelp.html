Import Images:

  • From your desktop, drag and drop the folder containing the images unto a folder on the left hand side of the window
  • You can also drag and drop individual images form your desktop or from one Folder on the left window to another.

Enter or select thumbnail Size (in pixels):

  • You need to hit the enter key after you enter a new maximum dimension or delete an old one from the combo box for the application to record the change.
Set the maximum dimension to use to generate thumbnails application wide.
Application generates thumbnails as follows. If the the max dimension of your image: (Jenn fix wording please..)
  • is < or equal to the selectedsize no thumbnail is generated.
  • is > specified size, then a thumbnail is generated based on the aspect ratio.
  • Aspect ratio = max dimension of your image / specified image size

Delete Images:

  • Select a folder or an image in the left window and select the 'Delete/File' button to delete it.

View Image:

  • A thumbnail image of a selected image appears in the right hand upper window.

Done:

  • Click on the 'Done' button to close window.
