Electronic Field Guide

 

The EFG website aims to provide a database management system for manipulating taxonomic information associated with the tree of life based on Linnaeus' model. It is intended for use as an educational research tool to assist users in accessing data regarding taxa. It will help biology researchers to identify species by performing iterative classification through elaboration of characteristics. For more details, please see the overview page or the EFG Project Page.
