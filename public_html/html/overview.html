An Overview of the Electronic Field Guide:

    Introduction:

    The EFG is intended for use as an educational and research tool to assist users in accessing data regarding taxa. It aims to provide a database management system for manipulating taxonomic information associated with the tree of life based on Linnaeus' model. There are many web-sites that discuss this model. For a biography of Carolus Linnaeus click here to go to the University of California Museum of Paleontology at Berkeley page. Also visit the links page that is part of this Website for more references.

    EFG will help biology researchers to identify species by performing iterative classification through elaboration of characteristics.

    EFG Features:

    • Browse: This feature provides an interface for the navigation of the Tree of Life. You could browse in the following two ways:
      • By Direct Taxon: To request information on a specific taxon by typing in its name.
      • By Branch Traversal: To traverse the tree of life using a graphical interface.
    • Search: Search provides an interface for searching the tree of life on the basis of the search criteria submitted by the user. The search criterias can be based on properties such as color and shape of wings.
    • Report Observ.: (WORK IN PROGRESS) An authorized user in the field can report Environmental observations about any taxon using this interface.
    • Review Observ.: (WORK IN PROGRESS) This feature provides interface to review all the reported environmental observations.
    • Report Species: (WORK IN PROGRESS) An authorized user can report an Unknown species. Unknown species are any observed life form not yet identified through Linnaeus' hierarchy, thus not belonging to the tree of life.
    • Review Species: (WORK IN PROGRESS) To review all the reported unknowns.
    • Links: This page lists the links to other related sites. The goal of this page is to provide an easy access to pages of interest.
